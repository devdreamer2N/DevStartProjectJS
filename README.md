# DevStartProjectJS



Posting Page — Projeto DevStart

Este repositório contém o projeto final da segunda etapa do curso DevStart, no qual desenvolvi uma página dinâmica capaz de criar posts semelhantes aos encontrados em redes sociais como Facebook e LinkedIn.

O usuário pode escrever um título e um conteúdo, enviar o formulário, e os dados são enviados para uma API aberta, simulando o fluxo real de criação de postagens.
Todo o comportamento dinâmico foi implementado com JavaScript, conforme exigido no desafio.
